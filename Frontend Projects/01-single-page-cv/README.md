How to Run

To view the project, simply open the HTML file in any web browser.

### Instructions:

1. **Clone or download** this repository to your local machine.
2. **Locate** the `index.html` file in the project folder.
3. **Open** the file using your preferred web browser:
   - Double-click the file, OR
   - Right-click → **Open with** → Choose browser (Chrome, Firefox, Edge, etc.)

### Alternative: Using VS Code
If you are using Visual Studio Code, you can use the **Live Server** extension:
1. Right-click on `index.html`.
2. Select **"Open with Live Server"**.
